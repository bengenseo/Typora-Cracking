# Typora
用于获取Typora序列号工具
前往官方下载安装-然后进行安装
![image](https://github.com/dioos886/Typora/assets/31064101/9c817a14-2650-4fa4-868f-31e2ccc327cc)

将工具
1. 放 Typora 根目录, cmd 运行 license-gen.exe, 复制生成的序列号
2. 运行 TyproaCrack.exe
3. 打开 Typora 粘贴生成的序列号
